## Introduction

### What is McLeave™?
McLeave™ is a comprehensive leave management system designed specifically for McDonald's crew members and managers. It streamlines the process of requesting, managing, and tracking employee leave requests.

### Key Benefits
- **Paperless Process**: All leave requests are submitted and managed digitally
- **Real-time Updates**: Instant notifications when requests are approved or denied
- **Color-Coded Organization**: Easy identification of different crew designations
- **WhatsApp Integration**: Send notifications directly to crew members
- **Analytics Dashboard**: Track leave patterns and manage crew availability
- **Mobile-Friendly**: Access the system from any device

## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Email address
- WhatsApp (for notifications)

### First Time Setup

#### For Store Managers
1. Navigate to the McLeave™ system URL
2. Click "Register as Admin"
3. Fill in the registration form:
   - Store name
   - Manager name
   - Email address
   - Password (minimum 6 characters)
4. Click "Register"
5. Sign in with your credentials

#### For Crew Members
1. Navigate to the McLeave™ system URL
2. Click on "Crew Portal"
3. Click "Register here"
4. Fill in the registration form:
   - Full name
   - Employee ID number
   - Email address
   - Phone number (with country code, e.g., +673XXXXXXX)
   - Select your designation
   - Select your store location
   - Create a password (minimum 6 characters)
5. Click "Register"
6. Wait for admin approval
7. Sign in once approved

## Admin Portal

### Dashboard

The admin dashboard provides an at-a-glance view of:
- Total leave requests
- Pending requests (requires action)
- Approved requests
- Denied requests
- Total crew members
- Blocked dates
- Today's requests

#### Quick Actions
- Click on any stat card to filter requests
- Navigate between tabs using the top menu

### Managing Leave Requests

#### Viewing Requests
1. Click on the "Requests" tab
2. Use filters to narrow down:
   - **All**: Show all requests
   - **Pending**: Show only pending requests
   - **Approved**: Show approved requests
   - **Denied**: Show denied requests
3. Use the search bar to find specific crew members or leave types

#### Processing a Request
1. Click on any request card to open details
2. Review the request information:
   - Crew member name and designation
   - Phone number
   - Leave type
   - Duration and dates
   - Reason provided
   - Submission timestamp (Brunei time)
3. Add an optional note in the "Manager Response" field
4. Choose one of three actions:
   - **✅ Approve**: Grant the leave request
   - **❌ Deny**: Reject the leave request
   - **⏳ Pending**: Keep as pending (default status)

#### Additional Actions
- **📱 Send WhatsApp Notification**: Send the decision to the crew member via WhatsApp (only for approved/denied requests)
- **🗑️ Delete Request**: Remove completed requests from the system (only for approved/denied requests)

#### Bulk Actions
1. Select multiple pending requests using checkboxes
2. Click "Approve Selected" or "Deny Selected"
3. Confirm the bulk action
4. All selected requests will be processed automatically

### Crew Management

#### Viewing Crew Members
1. Click on "Crew Management" tab
2. View the complete list showing:
   - Name
   - Designation (color-coded)
   - Email
   - Phone number
   - Status (Active/Pending)

#### Adding a Single Crew Member
1. Click "+ Add Crew" button
2. Fill in the form:
   - Full name
   - Email address
   - Phone number (optional)
   - Password (minimum 6 characters)
3. Click "Add Crew Member"
4. The crew member can now sign in

#### Bulk Upload via Excel
1. Click "📊 Upload Excel" button
2. Click "Download Template File" to get the correct format
3. Fill in the template with crew information:
   - Name (required)
   - Email (required)
   - Password (required)
   - Phone (optional)
4. Click "Select Excel or CSV File"
5. Choose your completed file
6. Wait for the upload to complete
7. Review the success/error message

#### Editing Crew Profiles
1. Find the crew member in the list
2. Click the "Edit" button
3. Update any of the following:
   - Full name
   - ID number
   - Email address
   - Phone number
   - Designation
4. Click "Update Profile"

#### Resetting Crew Passwords
1. Click "Edit" on the crew member
2. Scroll to "Reset Password" section
3. Enter a new password (minimum 6 characters)
4. Click "Reset Password"
5. Confirm the action
6. Inform the crew member of their new password

#### Activating/Deactivating Crew Members
- **Pending → Active**: Click "Approve" button
- **Active → Inactive**: Click "Deactivate" button
- Deactivated crew members cannot sign in or submit requests

#### Deleting Crew Members
1. Click the "Delete" button next to the crew member
2. Confirm the deletion
3. **Warning**: This will also delete all their leave requests

### Managing Blocked Dates

#### What are Blocked Dates?
Blocked dates are days when crew members cannot request leave, typically:
- Public holidays
- Store events
- Peak business periods
- Inventory days

#### Setting Blocked Dates
1. Click on "Block Dates" tab
2. Select dates on the calendar by clicking them
3. Selected dates turn red
4. Click again to unblock a date

#### Best Practices
- Set blocked dates at least 2 weeks in advance
- Communicate blocked dates to crew members
- Review and update monthly

### Analytics Dashboard

#### Viewing Analytics
1. Click on "Analytics" tab
2. View various metrics:
   - Leave requests by type
   - Requests by status
   - Trends over time
   - Crew availability

#### Exporting Data
1. Click "Export to CSV" button
2. Choose the data type:
   - All requests
   - Filtered requests
   - Crew member list
3. File downloads automatically
4. Open with Excel or Google Sheets

### Settings

#### Updating Login Credentials
1. Click on "Settings" tab
2. Enter current password
3. Enter new username
4. Enter new password (minimum 6 characters)
5. Confirm new password
6. Click "Update Credentials"

#### Store Profile
- Update store name
- Update manager name
- Update contact information

### Dashboard

The crew dashboard shows:

### Submitting a Leave Request

#### Step-by-Step Guide
1. Click on "+ New Request" tab
2. Select the leave type:
   - **Off Day**: Regular day off
   - **Annual Leave**: Vacation leave (deducted from balance)
   - **Morning Shift**: Half-day morning
   - **Afternoon Shift**: Half-day afternoon
   - **Birthday Leave**: Birthday celebration (deducted from balance)
3. Select date range using the calendar:
   - Click start date
   - Click end date
   - Red dates are blocked and cannot be selected
4. Enter reason for leave
5. Review the request summary
6. Click "Submit Request"

#### What Happens Next?
1. Request goes to "Pending" status
2. Manager receives notification
3. Manager reviews and responds
4. You receive notification of the decision
5. If approved and deductible, leave balance is updated

### Viewing Request Status

#### Request History
1. Click on "📋 My Orders" tab
2. View all your requests with:
   - Status indicator (Pending/Approved/Denied)

#### Request Details
1. Click on any request card
2. View complete details:
   - Your information
   - Leave type and duration
   - Reason submitted
   - Manager's response note (if any)
   - Status

#### Canceling Requests
- **Pending requests**: Can be canceled anytime
- **Approved/Denied requests**: Cannot be canceled (contact manager)

## Features Guide

### WhatsApp Notifications

#### Setting Up
1. Ensure crew member phone numbers include country code
2. Format: +673XXXXXXX (Brunei) or +60XXXXXXXXX (Malaysia)
3. Numbers must be active WhatsApp accounts

#### Sending Notifications
1. Open a request with approved/denied status
2. Click "📱 Send WhatsApp Notification"
3. WhatsApp web opens with pre-filled message
4. Review and send the message

#### Notification Template
```
Hi [Crew Name],

Your leave request has been [APPROVED/DENIED].

Leave Type: [Type]
Dates: [Start Date] to [End Date]

[Manager's note if provided]

- McLeave Management
```

### Color-Coded Designations

#### Why Color Coding?
- Quick visual identification
- Easy filtering and sorting
- Better organization of crew types
- Improved workflow efficiency

#### Using Color Codes
- All designation badges are color-coded throughout the system
- Request cards show crew designation with color
- Crew management table displays designation colors
- Colors consistent across all views

### Time Zone (Brunei Time)

#### What is Brunei Time?
- UTC+8 (Brunei Darussalam Time)
- All timestamps displayed in Brunei time
- Request submissions recorded in Brunei time
- Manager responses timestamped in Brunei time

#### Why Brunei Time?
- Consistent time reference for all users
- Avoids confusion from different time zones
- Matches local business hours

### Leave Balance Tracking

#### How It Works
1. Each crew member starts with 14 days annual leave
2. When Annual Leave or Birthday Leave is approved:
   - Days are calculated from date range
   - Balance is automatically deducted
   - Updated balance shows immediately
3. Balance is persistent across sessions
4. Cannot request more days than available balance

#### Balance Management Best Practices
- Check balance before submitting requests
- Plan leave in advance
- Balance resets annually (admin managed)

### Request Archive/Delete

#### When to Delete Requests
- Completed requests cluttering the view
- Old requests no longer needed
- Incorrect submissions

#### How to Delete
1. Only approved or denied requests can be deleted
2. Pending requests should be changed to approved/denied first
3. Deletion is permanent and cannot be undone
4. Manager can delete any approved/denied request

## Troubleshooting

### Common Issues and Solutions

#### Cannot Sign In

**Problem**: "Invalid email or password" error

**Solutions**:
1. Verify email address is correct
2. Check password (case-sensitive)
3. Click "Forgot Password" if available
4. Contact manager to reset password (crew members)
5. Ensure account is active (not deactivated)

**Problem**: "Your account has been deactivated"

**Solutions**:
1. Contact your manager
2. Manager needs to reactivate your account
3. Check if you're using the correct email

---

#### Registration Issues

**Problem**: "Email already exists"

**Solutions**:
1. You may already have an account - try signing in
2. Use a different email address
3. Contact manager if you forgot your password

**Problem**: "Cannot find store in dropdown"

**Solutions**:
1. Ask manager to register the store first
2. Refresh the page to reload store list
3. Contact system administrator

---

#### Leave Request Problems

**Problem**: Cannot select dates on calendar

**Solutions**:
1. Red dates are blocked - choose different dates
2. Ensure you're selecting future dates (not past)
3. Check if you have sufficient leave balance
4. Try clicking again - may need to select start date first

**Problem**: "Insufficient leave balance"

**Solutions**:
1. Check your remaining leave days in header
2. Reduce the number of days requested
3. Choose a leave type that doesn't deduct balance (Off Day, shifts)
4. Contact manager about leave balance

**Problem**: Request submitted but not showing

**Solutions**:
1. Refresh the page
2. Check "📋 My Orders" tab
3. Verify internet connection was stable during submission
4. Resubmit if necessary

---

#### Manager/Admin Issues

**Problem**: Cannot see crew member requests

**Solutions**:
1. Verify crew member is assigned to your store
2. Check if requests are filtered (remove filters)
3. Refresh the page
4. Verify crew member actually submitted requests

**Problem**: Bulk upload failed

**Solutions**:
1. Download and use the template file
2. Verify headers match exactly: Name, Email, Password, Phone
3. Check all required fields are filled
4. Ensure email addresses are valid and unique
5. Verify passwords are at least 6 characters
6. Remove any extra columns or formatting
7. Try uploading smaller batches

**Problem**: Cannot edit crew profile

**Solutions**:
1. Ensure crew member belongs to your store
2. Refresh the page
3. Try signing out and back in
4. Check internet connection

---

#### WhatsApp Notification Issues

**Problem**: WhatsApp doesn't open

**Solutions**:
1. Verify phone number includes country code (+673 or +60)
2. Check phone number is correctly formatted
3. Ensure WhatsApp Web is allowed in browser
4. Try different browser
5. Update crew member's phone number

**Problem**: Wrong number/crew member

**Solutions**:
1. Edit crew profile
2. Update phone number with correct format
3. Include country code
4. Remove any spaces or special characters

---

#### Data Export Issues

**Problem**: CSV file won't open

**Solutions**:
1. Ensure Excel or Google Sheets is installed
2. Try opening with different program
3. Check file downloaded completely
4. Try export again

**Problem**: Export shows no data

**Solutions**:
1. Verify you have requests/crew members to export
2. Remove any active filters
3. Refresh page and try again

---

#### Display/Interface Issues

**Problem**: Page not loading correctly

**Solutions**:
1. Clear browser cache and cookies
2. Try different browser
3. Check internet connection
4. Disable browser extensions
5. Try incognito/private mode

**Problem**: Mobile view looks broken

**Solutions**:
1. Rotate device to landscape
2. Update browser to latest version
3. Try different mobile browser
4. Zoom out if text is too large

**Problem**: Colors not showing correctly

**Solutions**:
1. Refresh the page
2. Clear browser cache
3. Check if browser supports CSS3
4. Try different browser

## FAQ

### General Questions

**Q: Is McLeave™ free to use?**
A: Contact your system administrator for pricing and licensing information.

**Q: Can I access McLeave™ from my phone?**
A: Yes! McLeave™ is fully responsive and works on all devices - phones, tablets, and computers.

**Q: What browsers are supported?**
A: Chrome, Firefox, Safari, and Edge (latest versions recommended).

**Q: Is my data secure?**
A: Yes, McLeave™ uses industry-standard security with Supabase authentication and encrypted connections.

**Q: Can I use this offline?**
A: No, an internet connection is required to submit and view requests.

---

### For Crew Members

**Q: How much annual leave do I have?**
A: Each crew member starts with 14 days annually. Check your balance in the header when signed in.

**Q: Can I cancel a leave request?**
A: Yes, but only if it's still pending. Contact your manager to cancel approved requests.

**Q: Why can't I select certain dates?**
A: Red dates on the calendar are blocked by management (holidays, busy periods, etc.).

**Q: What's the difference between leave types?**
A:
- **Annual Leave & Birthday Leave**: Deducted from your balance
- **Off Day, Morning/Afternoon Shift**: Not deducted from balance

**Q: How long does approval take?**
A: Depends on your manager, but most are reviewed within 24-48 hours.

**Q: Will I get notified when my request is approved?**
A: Your manager can send you a WhatsApp notification if your phone number is registered.

**Q: Can I request leave for past dates?**
A: No, only future dates can be selected.

**Q: What if I need emergency leave?**
A: Submit the request immediately and contact your manager directly.

**Q: How do I change my designation?**
A: Contact your manager - only managers can update crew designations.

**Q: Can I see other crew members' leave requests?**
A: No, you can only see your own requests. Managers can see all requests.

---

### For Managers/Admins

**Q: How many crew members can I add?**
A: There's no limit on the number of crew members per store.

**Q: Can I approve requests in bulk?**
A: Yes! Select multiple pending requests and click "Approve Selected" or "Deny Selected".

**Q: How do I export leave data?**
A: Click "Analytics" tab, then "Export to CSV". The file can be opened in Excel.

**Q: Can I undo an approval?**
A: Yes, open the request and change status to "Denied" or "Pending".

**Q: How do I handle leave balance corrections?**
A: Edit the crew member's profile to manually adjust their balance (if feature available).

**Q: What happens when I deactivate a crew member?**
A: They cannot sign in or submit requests, but their data remains in the system.

**Q: Can deleted requests be recovered?**
A: No, deletion is permanent. Only delete requests you're sure you don't need.

**Q: How do I set up recurring blocked dates?**
A: Manually select all dates on the calendar. Set them monthly or quarterly.

**Q: Can I manage multiple stores?**
A: Each manager account is associated with one store. Create separate accounts for multiple stores.

**Q: How do I reset a crew member's password?**
A: Click "Edit" on the crew member, scroll to "Reset Password", enter new password, and click "Reset Password".

**Q: What file format for bulk upload?**
A: Excel (.xlsx), .xls, or CSV files with Name, Email, Password, Phone columns.

**Q: Can I customize leave types?**
A: Contact system administrator for customization options.

**Q: How do I handle crew members who quit?**
A: Either deactivate or delete their account. Deactivation preserves history.

**Q: What's the difference between designation types?**
A: Designations are organizational categories with different color codes for easy identification. They don't affect permissions.

### Technical Questions

**Q: What is the time zone for timestamps?**
A: All timestamps are displayed in Brunei Time (UTC+8).

**Q: How is leave balance calculated?**
A: End date - Start date + 1 = Total days. For example, Jan 1 to Jan 3 = 3 days.

**Q: Can I integrate with payroll systems?**
A: Contact system administrator for integration options.

**Q: Is there an API available?**
A: Contact system administrator for API access and documentation.

**Q: How often should I backup data?**
A: Data is automatically backed up by Supabase. Export CSV files monthly for local backup.

**Q: What happens if I forget my password?**
A:
- Crew members: Contact your manager to reset
- Managers: Use password recovery or contact system administrator

**Q: Can I customize the colors for designations?**
A: Contact system administrator for customization options.

**Q: Is two-factor authentication available?**
A: Contact system administrator for security feature options.

## Glossary

**Admin**: Store manager or supervisor with full access to manage crew and requests

**Annual Leave**: Paid vacation time deducted from yearly balance

**Blocked Dates**: Days when leave requests cannot be submitted

**Bulk Action**: Processing multiple requests simultaneously

**Crew Member**: Employee who can submit and view their own leave requests

**Deductible Leave**: Leave types that reduce annual leave balance

**Designation**: Crew member role/position with color-coding

**Leave Balance**: Remaining days of annual leave available

**Pending Request**: Leave request awaiting manager review

**Status**: Current state of request (Pending, Approved, Denied)
